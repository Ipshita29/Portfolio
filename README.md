# Portfolio

This is my professional portfolio website, built with **React** and **Vite**, and styled with **Vanilla CSS**.

## Live Demo
Check out the live site here: [https://portfolio-rho-three-42.vercel.app](https://portfolio-rho-three-42.vercel.app)

## Tech Stack
- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Vanilla CSS
- **Deployment:** Vercel

## 💻 Getting Started
To run the project locally:
```bash
cd my-portfolio
npm install
npm run dev
```
